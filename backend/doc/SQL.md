# SQL 表结构设计

> 前期 mvp 使用 sqlite

## 数据表及其字段划分

- 基本数据表
  - 教学知识表
    - id
    - 知识点名称
    - 知识点概要
    - 知识点内容
  - 学生信息表
    - id
    - 姓名
    - 备注
- 模型数据记录表
  - 会话记录
    - id
    - 学生id
    - 知识点id
    - 会话标题
    - 最终评分
  - 对话记录
    - id(OpenAI API的chat_id)
    - 会话id
    - 推理内容(可选是否向学生开放)
    - 正文
    - 工具调用
  - 模型知识库
    - id
    - 对话id
    - 标题
    - 注释
    - 正文
    - 备注(是否存在错误)
  - 模型评价
    - id
    - 对话id
    - 评价
    - 备注

## 表细节

- 基本数据表
  - 教学知识表(knowledge)

    | 字段名   | 类型 | 是否主键/不可为空 | 外键 | 注释       | 备注 |
    | -------- | ---- | ----------------- | ---- | ---------- | ---- |
    | id       | int  | 主键              | 否   | 知识点编号 |      |
    | title    | text | 否                | 否   | 知识点标题 |      |
    | describe | text | 否                | 否   | 知识点概要 |      |
    | content  | text | 否                | 否   | 知识点内容 |      |

  - 学生信息表(stu_info)

    | 字段名  | 类型        | 是否主键/不可为空 | 外键 | 注释     | 备注 |
    | ------- | ----------- | ----------------- | ---- | -------- | ---- |
    | id      | int         | 主键              | 否   | 学生编号 |      |
    | name    | varchar(10) | 是                | 否   | 学生姓名 |      |
    | comment | text        | 否                | 否   | 备注     |      |

- 模型数据记录
  - 会话记录(session)

    | 字段名  | 类型 | 是否主键/不可为空 | 外键         | 注释         | 备注 |
    | ------- | ---- | ----------------- | ------------ | ------------ | ---- |
    | id      | int  | 主键              | 否           | 会话唯一标识 |      |
    | stu_id  | int  | 是                | stu_info.id  | 学生编号     |      |
    | know_id | int  | 是                | knowledge.id | 知识点编号   |      |
    | title   | text | 是                | 否           | 会话标题     |      |
    | score   | int  | 否                | 否           | 评分         |      |

  - 对话记录(chat)

    | 字段名            | 类型         | 是否主键/不可为空 | 外键       | 注释         | 备注                |
    | ----------------- | ------------ | ----------------- | ---------- | ------------ | ------------------- |
    | id                | varchar(128) | 主键              | 否         | 对话唯一标识 | OpenAI API 响应的id |
    | session_id        | int          | 是                | session.id | 会话编号     |                     |
    | reasoning_content | text         | 否                | 否         | 推理过程     | 学生可见性控制      |
    | content           | text         | 否                | 否         | 正文内容     |                     |
    | tools             | text         | 否                | 否         | 工具调用     |                     |

  - 模型知识库(model_knowledge)

    | 字段名   | 类型         | 是否主键/不可为空 | 外键    | 注释           | 备注                     |
    | -------- | ------------ | ----------------- | ------- | -------------- | ------------------------ |
    | id       | int          | 主键              | 否      | 知识库唯一标识 |                          |
    | chat_id  | varchar(128) | 是                | chat.id | 对话编号       |                          |
    | title    | text         | 是                | 否      | 知识标题       |                          |
    | describe | text         | 是                | 否      | 概要           |                          |
    | content  | text         | 是                | 否      | 正文内容       |                          |
    | comment  | text         | 否                | 否      | 备注           | 用于记录知识是否存在问题 |

  - 模型评价(model_evaluation)

    | 字段名     | 类型         | 是否主键/不可为空 | 外键    | 注释           | 备注 |
    | ---------- | ------------ | ----------------- | ------- | -------------- | ---- |
    | id         | int          | 主键              | 否      | 评价表唯一标识 |      |
    | chat_id    | varchar(128) | 是                | chat.id | 对话编号       |      |
    | evaluation | text         | 是                | 否      | 模型评价       |      |
    | comment    | text         | 否                | 否      | 备注           |      |
